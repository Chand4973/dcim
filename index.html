üìã Usage Instructions for NetBox
1. Enable Column

In NetBox device list:

  1. Go to Devices ‚Üí Devices
  2. Click "Configure Table"
  3. Add "OOB IP Ping Status" column
  4. Save your table configuration
2. Use Ping Feature

For devices with OOB IP:

  1. Locate device in the table
  2. Click "Ping" button
  3. Wait for result (Online/Offline)
  4. Button auto-refreshes after 5 seconds
3. Troubleshooting

If issues occur:

  1. Check browser console for errors
  2. Verify OOB IP is configured
  3. Ensure network connectivity
  4. Check Django logs for backend errors
